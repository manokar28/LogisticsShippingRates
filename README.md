# Simple Interest Calculator

A basic command-line shell script tool designed to calculate simple interest based on user inputs.

## Purpose
This project helps students and developers quickly compute annual financial interest yields given a principal amount, time duration, and fixed interest rate.

## How to Use
1. Make the script executable: `chmod +x simple-interest.sh`
2. Run the script: `./simple-interest.sh`
3. Enter the requested parameters when prompted.

## Code Snippet Example
```bash
# Standard interest formula used:
interest=\((expr\)p \* \(r \*\)t / 100)
```
